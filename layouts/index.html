ALF Association


We are a registered non-profit with the goal of supporting the Algorithms for Lattice Fermions (ALF) project.

We dedicate our funding for the package's:
  • maintenance
  • improvement and
  • user base support and growth
The ALF package provides an open-source, high-performance, general code for auxiliary-field quantum Monte Carlo simulations.
